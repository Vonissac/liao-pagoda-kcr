# Data Availability

The public repository provides the data required to reproduce the manuscript's reported KG retrieval, CBR, LPSL and CoT-ablation summaries. It includes normalized LP-KG export tables, KG source/evidence registry, benchmark inputs and outputs, CBR inputs and rankings, LPSL encodings and validation records, model-input instructions, raw model outputs, metric workbooks and supplementary source tables.

Raw page images and tower-named elevation images derived from Wang Zhuonan's published Digital Liao Pagoda materials are not redistributed here because public repository rights must be confirmed separately. The repository therefore includes the measurement workbooks and release-boundary documentation, while omitting image binaries from the public release layer.

The repository is intended for later DOI release after repository review.
