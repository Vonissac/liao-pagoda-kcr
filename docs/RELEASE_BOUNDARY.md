# Public Release Boundary

Included in this public repository:

- normalized full LP-KG node and relationship exports;
- KG source/evidence registry distinguishing public structured inputs from restricted source carriers;
- released public project-KG execution tables;
- KG retrieval benchmark inputs and recorded outputs;
- CBR inputs, rankings and pagoda attribute corpus;
- LPSL encodings, validation replay table and measurement workbooks;
- CoT-ablation model-input instructions, source texts, raw model outputs and metric workbooks;
- scripts required for deterministic local reproduction.

Excluded from this public package:

- unpublished source-transcription materials and extraction tables outside the released data boundary;
- system files;
- raw page images and tower-named elevation images whose public repository redistribution requires separate rights confirmation;
- materials outside the reported analysis.
