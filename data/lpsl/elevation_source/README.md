# Elevation Image Boundary

The public repository keeps reproducible tabular summaries and measurement workbooks. Raw page images and tower-named elevation images derived from Wang Zhuonan's published Digital Liao Pagoda materials are not redistributed here because repository rights must be confirmed separately.

If image binaries are later cleared for public redistribution, they can be placed under `named_elevation_views_wang_zhuonan/` with an updated rights note and data manifest.
